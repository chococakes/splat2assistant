In order to view or update your battle stats, you must have your NSOnline Splatoon 2 cookie - also known as the iksm_session. To get that, you have to use a web debugging/capturing tool such as mitmproxy or Fiddler.

Use the guide here to get the cookie using Fiddler! Once you have the iksm_session cookie, type it in here and view your stats! The cookie should be forty characters long.

If you've already used this site to view stats, you should have been given an ID on the viewer page. You can also enter that here to view saved data! Keep in mind that you need the iksm_session cookie to update the data with your latest battles. Your ID should be sixteen characters long.

Give it a try with the ID "eb25cb0e499d5911".

Add me on Discord if you'd like - I'm DestinyWaffle#1009.

Hello, I'm Choco. I'm just a fourteen year old girl who likes inking turf. This has been my first site, and it's been quite the learning experience.

Since I love Splatoon, I figured I would make something that could show me how well/terribly I play. I made a python console application and started to leave it as that, but then I decided to give web development a shot, since I'd never really done it before. What better idea to start with? I didn't exactly think it would be easy, but I didn't think it would be that hard.

Was I wrong? Maybe.

I don't really have the best eye for design. Learning how to position elements, learning how to use the Flask framework, learning how to use templates... It's been something. I've gotten most of my information from Google - I haven't really asked for help from actual people except for the odd small question on Discord. I looked at other Splatoon sites, such as splatoon2.ink and loadout.ink for a design guideline. Without those sites, I probably would have been even more clueless.

Anyway, I'm still proud of it. It's one of the first times I've set a goal and actually did something close to ""finishing it"" - although I'm still looking for ways to improve it - so it's quite the personal accomplishment for me. It may not look the best, but it's my best so far, and I'm happy.

Oof. This got kind of rambly. Anyway, thanks for using the site! I hope it's useful to you :>
